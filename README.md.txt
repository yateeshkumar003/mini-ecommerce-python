# 🛍️ Mini eCommerce System - Python

This is a simple console-based eCommerce simulation project using **Object-Oriented Programming** in Python.

## 🚀 Features
- Add products (Grocery, Electronics, Laptops)
- Display full product details
- Add items to cart and place an order
- Dynamic delivery charges
- Bill and savings calculation

## 🧠 Concepts Used
- Python Classes and Inheritance
- Method Overriding
- Class vs Instance Variables
- Constructor chaining (super())
- Clean, modular code design

## 📦 How to Run

```bash
python ecommerce.py
